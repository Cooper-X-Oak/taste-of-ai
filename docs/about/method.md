# 评测方法论

每期固定流程:

`开胃(选题) → 出题(标准考题) → 试吃(逐个跑·留证) → 评级(六维卡+梯队) → 主厨锐评 → 一句话外带`

## 出题规范(保证公平)

- **横评**:全擂台共用**同一道标准考题**(同一个真实任务、同样输入),才能同锅对比。考题锁死后不改。
- **竖评**:设 3–5 个递进场景(从宣传卖点打到刁钻边界)。

## 留证规范(L1 证据)

每个选手记录:

1. 完整 prompt / 调用方式(skill 必须用 Skill 工具真调用,不靠记忆模拟)
2. 跑了几轮、有没有翻车、怎么救
3. 关键产出(截图 / 文件 / 链接,存 `evidence/`)
4. 耗时、token 体感
5. **菜单(官方描述)vs 实物(实测)** 对照

> 铁律:没有 L1 证据的判断只是 L3 先验,不许当结论写进手册。
