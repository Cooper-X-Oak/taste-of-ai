# 评级体系 · 从夯到拉

## 六维评分卡(每维 ⭐1–3)

| 维度 | 美食黑话 | 实测什么 |
|---|---|---|
| 上菜速度 | 出餐快不快 | 响应 / 产出速度 |
| 火候 | 生不生熟不熟 | 完成度,能不能直接端上桌(开箱可用) |
| 口感 | 好不好吃 | 输出质量,有没有惊艳点 |
| 回甘 | 后劲香不香 | 可复现 / 可维护,过一阵还顶不顶用 |
| 摆盘 | 卖相 | 默认产物的美观 / 规整(不像预制菜套模板) |
| 菜单 vs 实物 | 照骗指数 | 宣传(L3 先验)对照实测(L1 后验),差距越大扣越狠 |

满分 18⭐。**打分必须挂 L1 证据,否则不计。**

## 段位:从夯到拉

| 段位 | 黑话 | 含义 | 参考总⭐ |
|---|---|---|---|
| 🔥 S | 夯爆 · 招牌硬菜 | 真香,装了回不去 | 16–18 |
| 😋 A | 够味 · 家常好菜 | 稳,值得常备 | 13–15 |
| 😐 B | 垫吧 · 能下饭 | 凑合,有平替看场景 | 10–12 |
| 🥱 C | 预制菜 · 照骗 | 菜单天花乱坠,入口塑料味 | 7–9 |
| 🤮 D | 拉胯 · 黑暗料理 | 避雷,别碰 | ≤6 |

⭐ 为参考,主播锐评可微调段位(需说明理由)。

## 主厨锐评写法

- 一句话定生死,毒舌但**有据**(引用实测里的具体表现)。
- 必点明「菜单 vs 实物」的反差。
- 给出「谁该点这道菜 / 谁别碰」。
