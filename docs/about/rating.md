# 评级体系 · 从夯到拉

凭一个普通用户「好不好用」的个人体验下结论,**不算分、不搞同尺打分**,直接归档到「从夯到拉」五档(网络通用 tier 黑话):

| 段位 | 词 | 含义 |
|---|---|---|
| 🔴 S | 夯 | 强到离谱,装了回不去 |
| 🟠 A | 顶级 | 优秀出众,值得常备 |
| 🟢 B | 人上人 | 整体不错,部分能打 |
| ⚪ C | NPC | 平庸路人,可有可无 |
| ⚫ D | 拉(完了) | 拉胯,避雷别碰 |

## 照骗不是一档

"官方吹得天花乱坠 vs 实测拉胯"的反差(照骗 / 名不副实)是单独的点评,不占段位 —— 它写在文章的「菜单 vs 实物」里,顺带把段位拖下去。避雷红榜只收 **D(拉)**。

## 每个 skill 的结论长这样

- **小白试吃**:一句话(模糊提示词下好不好用)
- **专家试吃**:一句话(清晰提示词下上限如何)
- **最佳用法**:这么用最对
- **总评**:好不好用一句斩 + 段位 + 谁该装 / 谁别碰
