# 评级体系 · 从夯到拉

凭一个普通用户「好不好用」的个人体验下结论,**不算分、不搞同尺打分**,直接归档:

| 段位 | 黑话 | 含义 |
|---|---|---|
| 🔥 S | 夯爆 · 招牌硬菜 | 真香,装了回不去 |
| 😋 A | 够味 · 家常好菜 | 稳,值得常备 |
| 😐 B | 垫吧 · 能下饭 | 凑合,有平替看场景 |
| 🥱 C | 预制菜 · 照骗 | 宣传天花乱坠,实际拉胯 |
| 🤮 D | 拉胯 · 黑暗料理 | 避雷,别碰 |

## 每个 skill 的结论长这样

- **小白试吃**:一句话(模糊提示词下好不好用)
- **专家试吃**:一句话(清晰提示词下上限如何)
- **最佳案例**:这么用最香
- **总评**:好不好用一句斩 + 段位 + 谁该装 / 谁别碰

> 主观个人意见,仅供参考。但每条都挂实测证据,不瞎说。
